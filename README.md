# dsPIC morse code converter

シリアル通信で文字列を受信し，モールス信号に変換してブザーを鳴らすシステム

## システム概要
- **受信内容**  
  シリアル通信を通じて文字列を受信．
- **処理内容**  
  受信した文字列をモールス信号に変換し，対応する音声信号を生成してブザーを駆動．

## 使用機器
- **マイコン**: dsPIC30F6012A  
- **ブザー**: 標準的なアクティブブザー

## 開発環境
- **統合開発環境 (IDE)**: MPLAB.X  
- **コンパイラ**: XC16  
- **プログラム書き込みツール**: ICD3  

## ハードウェア構成
1. **UARTピン**: シリアル通信を行うためにPCまたは他のシステムと接続．
2. **ブザーピン**: モールス信号の音声出力用にGPIOポートを接続．

## ソフトウェアの仕組み
1. シリアル通信で文字列を受信します。
2. 各文字を対応するモールス信号に変換します。
3. モールス信号に従ってGPIO出力を切り替え，ブザーを駆動します。

## 使用方法
### 1. ビルドと書き込み
1. MPLAB.Xを開き、本プロジェクトをインポート．
2. XC16コンパイラでビルドを実行．
3. ICD3を使用してdsPIC30F6012Aにプログラムを書き込む．

### 2. 実行
1. シリアル通信ソフトウェア（例: Tera Term）を使用して，マイコンと接続．
2. 任意の文字列を送信．
3. ブザーから対応するモールス信号音が出力される．
